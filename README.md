Онлайн задачник по высшей математике "Демидович" с ответами, подробными решениями и справочными материалами.

* Максимально подробный разбор каждой задачи
* Возможность посмотреть указание и итоговый ответ
* Выделение типовых приемов решения в прото-задачи
* Тематическая разбивка задач внутри параграфов
* Быстрый доступ к задаче по ее номеру
* Поддержка альтернативных решений от других авторов

## Локальная разработка

1. Установите Node.JS
2. Установите ПО для сборки нативных Node.JS модулей. На Windows для этого запустите файл `install_tools.bat` по пути `C:/Program Files/nodejs`
3. Скопируйте к себе на компьютер этот GitHub репозиторий
4. Находясь в директории с скопированным проектом выполните команду `npm install`

Все готово!

Если вы хотите просматривать изменения в реальном времени, введите следующую команду в терминале:

```bash
npx ts-node src/scripts/watch --task=1
```
Тогда изменения можно будет смотреть на странице `http://localhost:3000/tasks/1/`.
Не забудьте поменять номер задачи на ту, над которой вы работаете.
